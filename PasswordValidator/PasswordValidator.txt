1)
Método evaluarPassword evalúa la seguridad de una contraseña en base a varios criterios.

Entradas:
String password

Salidas:
String:
- "DEBIL"
- "MEDIA"
- "FUERTE"

Criterios:
+1 si tiene al menos 8 caracteres
+1 si tiene mayúsculas
+1 si tiene minúsculas
+1 si tiene números
+1 si tiene símbolos

Casos:
- contraseña muy débil
- contraseña media
- contraseña fuerte

2)
Número de decisiones:
length >= 8 => 1
mayúsculas => 1
minúsculas => 1
números => 1
símbolos => 1
clasificación final => 2

Total: 7

TM = 7 + 1 = 8
